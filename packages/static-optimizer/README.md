# @threeforged/static-optimizer

Optimize static geometry in Three.js scenes by merging non-moving meshes to reduce draw calls and improve rendering performance.

> **Status:** Coming soon. This package is currently a placeholder.

## Installation

```bash
pnpm add @threeforged/static-optimizer
```

## Planned Features

- Static mesh detection and merging
- Geometry batching by material
- Buffer geometry optimization
- Vertex deduplication
- Bounding volume generation
