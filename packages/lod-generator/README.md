# @threeforged/lod-generator

Generate levels of detail (LOD) for Three.js meshes. Automatically creates simplified mesh variants for distance-based rendering.

> **Status:** Coming soon. This package is currently a placeholder.

## Installation

```bash
pnpm add @threeforged/lod-generator
```

## Planned Features

- Automatic mesh simplification at configurable quality levels
- LOD group generation for Three.js LOD objects
- Triangle budget targeting
- UV and attribute preservation
- Batch processing for entire asset directories
