# @threeforged/performance-auditor

Audit Three.js scene performance. Analyzes draw call patterns, overdraw, shader complexity, and runtime bottlenecks.

> **Status:** Coming soon. This package is currently a placeholder.

## Installation

```bash
pnpm add @threeforged/performance-auditor
```

## Planned Features

- Draw call analysis and batching recommendations
- Shader complexity scoring
- Overdraw detection
- Material and geometry deduplication suggestions
- Scene graph depth analysis
- Performance budget enforcement
